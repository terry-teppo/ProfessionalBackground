## U.S. Navy Data Processing Technician Training

**1989–1990**

### Phase I — "A" School Core Technical Foundations (1989)

#### Binary, Octal, and Hexadecimal Systems

**Focus:** Number system conversions and their application in computing

**Skills Covered:**

- Conversion between binary, octal, hexadecimal, and decimal
- Application in memory addressing and machine instructions
- Understanding how computers process and store information at the bit level

**Notes:** Provided the mathematical foundation for programming and systems operations.

#### Card Punching and Batch Processing

**Focus:** Legacy data entry and batch job management

**Skills Covered:**

- Operation of punched card systems for program loading and data entry
- Sequencing batch jobs for large-scale data processing
- Error detection and correction in physical media

**Notes:** Reinforced knowledge of early computing methods still in use for archival and transitional systems.

#### Magnetic Tape Systems Operation

**Focus:** Storage and retrieval using magnetic tape drives

**Skills Covered:**

- Loading, mounting, and maintaining tape drives
- Sequential data access and archival storage
- Backup and recovery procedures using tape media

**Notes:** Essential for managing Navy data archives and logistics systems.

#### Systems Operations

**Focus:** Fundamentals of mainframe and minicomputer environments

**Skills Covered:**

- Job control languages and scheduling
- System security and data integrity
- Monitoring and troubleshooting system operations

**Notes:** Prepared sailors to manage mission-critical computing environments afloat and ashore.

### Phase II — Advanced Programming & PC Training (1989–1990)

#### dBase III & IV Programming

**Focus:** Database programming and management using dBase III and IV

**Skills Covered:**

- Relational database design and schema creation
- Querying, indexing, and reporting for logistics and maintenance records
- Procedural programming with dBase language constructs
- File handling, batch processing, and automation of repetitive tasks

**Notes:** Provided foundational database programming skills, directly applicable to Navy maintenance and inventory systems.

#### COBOL Programming I & II

**Focus:** Business-oriented programming for data processing and reporting

**Skills Covered:**

- Structured COBOL syntax and program design
- File handling, batch processing, and record management
- Report generation for financial, personnel, and logistical applications
- Error handling and debugging in large-scale programs

**Notes:** Reinforced skills in legacy business programming, aligning with Navy administrative and financial systems of the era.

#### C Programming

**Focus:** Structured programming and systems-level development in C

**Skills Covered:**

- Syntax, data types, and control structures
- Modular programming and function libraries
- Memory management and pointer operations
- File I/O and system calls for low-level operations

**Notes:** Introduced modern programming practices, preparing for later work in systems administration and enterprise software environments.

#### PC Systems Training (Lower-Level Class)

**Focus:** Introduction to IBM-compatible PCs and DOS environments

**Skills Covered:**

- DOS command line operation, file systems, and utilities
- Using PCs as terminals connected to larger naval systems
- Early exposure to distributed computing environments

**Notes:** Provided sailors with practical PC skills as the Navy began transitioning from centralized mainframes to distributed desktop systems.
